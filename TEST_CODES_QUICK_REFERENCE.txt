TROUBLESHOOTING:
================================================================================

Problem: "Invalid access code" error
Solution: 
  1. Check if code was inserted into database (Supabase → access_codes table)
  2. Verify code is exactly 8 characters
  3. Try a different code
  4. Clear browser cache and try again

Problem: Can't access dashboard
Solution:
  1. Check Supabase connection (check env variables)
  2. Verify business was created successfully
  3. Check browser console for errors (F12)
  4. Clear cookies and try again

Problem: Data not saving
Solution:
  1. Open console (F12) and check for errors
  2. Check Network tab for failed requests
  3. Verify Supabase is connected
  4. Check RLS policies are configured

Problem: Charts not displaying
Solution:
  1. Add sales/expense records first
  2. Refresh the page
  3. Clear browser cache
  4. Check console for JavaScript errors
