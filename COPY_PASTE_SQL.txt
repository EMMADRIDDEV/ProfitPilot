COPY-PASTE THIS SQL INTO YOUR SUPABASE SQL EDITOR
================================================

Step 1: Create the access_codes table
Go to: https://app.supabase.com → SQL Editor → New Query
Copy and paste this exact SQL:

---
CREATE TABLE IF NOT EXISTS public.access_codes (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  code VARCHAR(8) UNIQUE NOT NULL,
  user_id UUID,
  is_used BOOLEAN DEFAULT FALSE,
  expires_at TIMESTAMP WITH TIME ZONE DEFAULT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
---

Click RUN and wait for success.

Step 2: Create the users table
Create another new query and paste this:

---
CREATE TABLE IF NOT EXISTS public.users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  email VARCHAR(255) UNIQUE,
  is_verified BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
---

Click RUN and wait for success.

Step 3: Create the businesses table
Create another new query and paste this:

---
CREATE TABLE IF NOT EXISTS public.businesses (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL,
  business_name VARCHAR(255) NOT NULL,
  business_email VARCHAR(255),
  business_phone VARCHAR(20),
  business_address TEXT,
  business_type VARCHAR(100),
  registration_number VARCHAR(100),
  tax_id VARCHAR(100),
  industry VARCHAR(100),
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
---

Click RUN and wait for success.

Step 4: Create the sales table
Create another new query and paste this:

---
CREATE TABLE IF NOT EXISTS public.sales (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  business_id UUID NOT NULL,
  sale_date DATE NOT NULL,
  product_name VARCHAR(255) NOT NULL,
  quantity INTEGER NOT NULL,
  unit_price DECIMAL(10, 2) NOT NULL,
  total_amount DECIMAL(10, 2) NOT NULL,
  payment_method VARCHAR(50),
  notes TEXT,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
---

Click RUN and wait for success.

Step 5: Create the inventory table
Create another new query and paste this:

---
CREATE TABLE IF NOT EXISTS public.inventory (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  business_id UUID NOT NULL,
  product_name VARCHAR(255) NOT NULL,
  product_code VARCHAR(50),
  category VARCHAR(100),
  quantity_in_stock INTEGER NOT NULL,
  reorder_level INTEGER,
  unit_cost DECIMAL(10, 2),
  selling_price DECIMAL(10, 2),
  supplier_name VARCHAR(255),
  supplier_contact VARCHAR(20),
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
---

Click RUN and wait for success.

Step 6: Create the expenses table
Create another new query and paste this:

---
CREATE TABLE IF NOT EXISTS public.expenses (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  business_id UUID NOT NULL,
  expense_date DATE NOT NULL,
  category VARCHAR(100) NOT NULL,
  description TEXT,
  amount DECIMAL(10, 2) NOT NULL,
  payment_method VARCHAR(50),
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
---

Click RUN and wait for success.

That's it! All tables are now created in your Supabase database.
Now go back to the app and try generating an access code again.
